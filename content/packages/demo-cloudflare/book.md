# @f12o/papyr-demo-cloudflare

`@f12o/papyr-demo-cloudflare` は、Papyr を Cloudflare Workers 上でどう公開アプリに組み立てるかを示す参考実装です。public pages と studio を同じ SPA に載せ、保存先には `@f12o/papyr-adapter-cloudflare` を使います。

「Papyr の package は分かったが、実際に article / book / chapter をどう運ぶか」を見るためのサンプルとして位置づけると分かりやすいです。
