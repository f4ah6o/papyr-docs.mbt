## 実装メモ

Moonlight は MoonBit で書かれた SVG editor なので、Papyr 側では編集用の構造を再実装せず、保存形式を `svg` 文字列に寄せた。`svg` は Moonlight editor の `exportSvg()` で取り出せる完成形の markup で、preview 側ではこの SVG を検査してから DOM に入れる。

`papyr-moonlight` fence は JSON payload として `{ "svg": "...", "caption": "..." }` を保存する。これは Markdown には標準の図形編集構造がないため、Papyr 独自 block として再編集に必要な情報を残すための形式。

preview では `<script>` や `onclick` のような event handler 属性を落とす sanitizer を追加した。sanitizer は「外部から来た文字列をそのまま HTML として解釈しない」ための安全弁で、空の SVG と壊れた SVG はそれぞれ空状態・エラー状態として表示する。
